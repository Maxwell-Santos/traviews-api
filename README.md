# Traviews-api
O traviews foi criado com o objetivo de centralizar o compartilhamento de viagens e recomendações entre as pessoas.

## Tópicos que procurei abordar na criação dessa api
- Usar arquitetura modular com DDD
- Repositositorie Pattern
- SOLID
- POO
- Arquitetura Monolítica modular: A arquitetura monolítica modular é uma forma de unir a simplicidade de operação de um monólito com a organização e clareza de um sistema modular. Escolha estratégica para esse sistemas em estágio inicial e aceita tranquilamente um crescimento sustentável.
